# AU-Canteen-Management
A university canteen site for online food ordering and automated token generation.
Main Features
Menu: View available food items and prices.

Ordering: Select items and place an order.

Token System: Automatically generates a unique token/ID for every order.

Admin Panel: Simple view for canteen staff to see incoming orders and mark them as done.

How to Use
Login with your student ID.

Select your food from the menu.

Confirm your order.

Show the Token at the counter to collect your food.

Tech Used
Frontend: HTML, CSS, JavaScript
